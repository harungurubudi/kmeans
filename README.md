# Kmeans

Implementasi K-Means for data clustering

### Requirement
- php7.2
- mysql based database
- composer

### Installation
- ``composer install``
- Tambahkan file ``.env`` dengan isi sebagai berikut
```
ENV="<dev | prod>"
```

- Atur file konfigurasi pada ``config/`` sesuai dengan environment
- Siyap development
